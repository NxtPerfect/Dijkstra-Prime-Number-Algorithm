# Dijkstra Prime Number Algorithm
dijkstra's algorithm for finding prime numbers implemented in typescript

# Usage
```
dijkstra(howManyNumbersToCheckForPrime) # Returns array of all primes
```

# Performance
| n | time took |
| -------------- | --------------- |
| 100 | 0ms |
| 1 000 | 21ms |
| 1 000 000 | 1m49s |
